= Changelog
:description: This chapter lists the notable changes for each release of the Neo4j Connector for Apache Spark, mirroring the GitHub release entries.

This page mirrors the link:https://github.com/neo4j/neo4j-spark-connector/releases[GitHub releases^] of the Neo4j Connector for Apache Spark, listing headline features, fixes, and breaking changes for every minor and patch release of the 5.x line.

[#_5_4_0]
== Version 5.4.0 (12 Jan 2026)

[cols="1,2",options="header"]
|===
| Feature | Details

a|
label:functionality[]
label:breaking[]

Type conversion default changed.
| Timestamps, intervals, and byte arrays are now converted differently from previous releases. Set the `type.conversion` reader option to `legacy` to keep the pre-5.4.0 behavior. See xref:types.adoc[] and xref:migration.adoc#_5_3_to_5_4[Migration 5.3 → 5.4].

a|
label:functionality[]
label:breaking[]

Strict `relationship.properties` map.
| When `relationship.properties` is set, only the listed properties are written; previously, unmapped fields were written too. See xref:write/options.adoc[] and xref:migration.adoc#_5_3_to_5_4[Migration 5.3 → 5.4].

a|
label:feature[]
label:new[]

Native interval support.
| Spark intervals are now mapped to and from the Neo4j `Duration` type. See xref:types.adoc[].

a|
label:feature[]
label:new[]

New `type.conversion` reader option.
| Opt back into the legacy conversion logic. See xref:read/options.adoc[].

a|
label:feature[]
label:new[]

Global await for index creation.
| Schema optimization can now await index creation globally. See xref:write/schema-optimization.adoc[].

a|
label:bug[]
label:fixed[]

Return `null` when no last offset is available while streaming.
| See xref:streaming.adoc[].

|===

label:changed[]

Documentation extracted to a separate repository.
| The documentation was extracted to a separate repository at link:https://github.com/neo4j/docs-spark/[neo4j/docs-spark^].

== Version 5.1.0 (30 Aug 2023)

[cols="1,2",options="header"]
|===
| Feature | Details

a|
label:functionality[]
label:breaking[]

Spark baseline raised to 3.3.
| This release requires Apache Spark 3.3 or later in order to support push-down v2 filters and limit optimizations. 5.1.0 is the last release compatible with Spark 3.3. See xref:installation.adoc#version[the compatibility table] and xref:migration.adoc#_5_0_to_5_1[Migration 5.0 → 5.1].

label:new[]

Push-down v2 filter support.
| Controlled by the `pushdown.filters.enabled` reader option. See xref:read/options.adoc[].

a|
label:feature[]
label:new[]

Push-down LIMIT support.
| Controlled by the `pushdown.limit.enabled` reader option. See xref:read/options.adoc[].

a|
label:feature[]
label:new[]

First-class support for Graph Data Science (GDS).
| Adds first-class support for xref:gds.adoc[Graph Data Science (GDS)].

a|
label:feature[]
label:new[]

Bearer authentication support.
| Adds support for bearer authentication. See xref:configuration.adoc[].

a|
label:bug[]
label:fixed[]

Handling of datetime properties that have `null` values.
| Fixes handling of datetime properties that have `null` values.

a|
label:bug[]

Spark version validation.
| Fixes Spark version validation.

|===
